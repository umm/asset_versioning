# Asset Versioning

## What

* Set asset version into `AssetImporter.userData` field

## Requirement

* 

## Install

```shell
yarn add "umm/asset_versioning#^1.0.0"
```

## Usage

1. Selection some assets in Project View
1. Run `Assets > Versioning > Increment`
